# DELMA-CLICKERHEHE
Delma Clicker to prosta, ale wciągająca gra przeglądarkowa typu clicker, w której Twoim głównym zadaniem jest… klikanie! Każde kliknięcie daje punkty, które możesz później wykorzystać na zakup ulepszeń.
